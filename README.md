  Learning Is Fun (Static)

  Playful, kid-friendly English practice built as a single static HTML file. The page includes tabs for letters,
  flashcards, sentence building, and a simple word lab with audio feedback.

  Features

  - Letters + sounds with spoken prompts
  - Flashcards with phonics and example sentences
  - Sentence builder with word bank and "surprise" sentences
  - Word Lab to blend onset/vowel/coda sounds
  - Lightweight audio feedback using Web Speech + Web Audio APIs

  Quick start

  1) Clone the repo:

  git clone https://github.com/hafsaghannaj/learning-is-fun-static.git
  cd learning-is-fun-static


  2) Open the site:
  - macOS: `open index.html`
  - Windows: `start index.html`
  - Linux: `xdg-open index.html`

  ## Project structure

  - `index.html` - All HTML, CSS, and JavaScript (no build step)
  - `README.md` - Project documentation

 Notes

  - Audio features rely on browser support for `SpeechSynthesis` and `AudioContext`.
  - Designed for preschool to grade 3 learners.
